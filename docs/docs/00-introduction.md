# Getting Started 🚀

Welcome to the AI Agent Workshop!

![lab-summary.png](../media/lab-summary.png)

## What Will You Learn?

- The basics of agentic AI and how agents differ from traditional models
- How to set up your Azure environment
- How to build your first agent: define goals, tools, and memory
- How to test and interact with your agent
- Advanced features like tool integration and memory management

## Who Is This For?

This workshop is designed to help you build AI agents using Semantic Kernel and Azure AI Foundry. Whether you're a beginner or an experienced developer, you'll find valuable insights and hands-on experience throughout the labs.

## How Does the Workshop Work?

- Step-by-step guides at your own pace

The labs are meant to be completed in order, as each builds on the knowledge and skills acquired in the previous one.
